# Enterprise Data Platform

Open-source modules for building a data engineering platform on AWS.

The repositories demonstrate:
- Change Data Capture (CDC) with AWS DMS
- Medallion architecture (Bronze, Silver, Gold layers)
- Data transformation with Glue, dbt, and Apache Iceberg
- Orchestration using MWAA (Airflow)
- Natural language analytics Agent with Athena and Claude

Detailed documentation and guides are available in the individual repository README files.
